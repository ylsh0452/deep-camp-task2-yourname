**用 AI 工具链（Claude Code + DeepSeek V4 Pro）从零搭建个人技术博客，产出 3 篇 AI 辅助写作的原创技术文章，记录完整的 AI 协作工作流。**

## 📖 任务概述

DEEP 营任务二要求：选择一个真实场景，明确具体问题，设计 AI 工具使用流程，完成最终交付物，并说明 AI 在这个过程中到底提升了什么、有哪些局限、哪些判断必须由自己完成。

### 选择的场景

**搭建个人技术博客并产出 3 篇技术文章。**

### 选择的理由

我的任务一个人壁垒路线图中明确写了「建立技术输出：写技术笔记/博客/复盘，累计 10+ 篇」，但到目前为止为 0。这不是"假设的需求"，而是路线图上写了、还没开始做的事。任务二恰好是启动这个目标的最佳时机。

### 真实问题

1. 需要一个可访问的个人博客网站来承载技术输出
2. 需要产出第一批高质量的原创内容
3. 需要学会 AI 辅助写作的工作流（不是让 AI 代写，而是用 AI 提升写作效率和质量）
